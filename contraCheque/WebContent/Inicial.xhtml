Instruções

1. Siga o passo a passo e, ao concluir cada um, vá para próxima etapa

2. Clique no botão abaixo até que ele indique para ir para o próximo passo

3. No passo 2 você irá anexar o arquivo "EXPORTA.TXT" gerado pero programa do RH (\\Servidor\rh\CCS\temp)

4. Escolha se deseja apenas gerar os contracheques ou gerar e publicar diretamente no site

5. Visite o site e faça a verificação final, faça login com seu usuário e senha para executar este passo
